## Equalizers

- **FabFilter Pro-Q 3 / Pro-Q 4** — Industry-standard parametric EQ with dynamic EQ, linear phase, spectrum analyzer, and M/S modes. The go-to surgical EQ.
- **FabFilter Micro / Simplon** — Lightweight, focused filter/EQ plugins for quick tonal shaping.
- **Carve EQ** — Spectral EQ from iZotope for broadstroke tonal balancing.
- **Dragon EQ** — Creative EQ with character.
- **Slice EQ** — Precision surgical EQ tool.
- **kHs 3-Band EQ** — Kilohearts snap-in 3-band EQ for fast workflow.
- **SPL Free Ranger** — 4-band passive-style EQ for musical, analog-flavored shaping.
- **elysia niveau filter** — Tilt EQ for broad, natural tonal shifts.
- **SPAN** — Free real-time FFT spectrum analyzer by Voxengo. Essential reference tool.
- **SmoothOperator** — Spectral resonance suppressor and tonal smoother.
- **TEOTE** — That Exciting Opinionated Tonal Equalizer — automatic broad tonal balancing.
- **Ozone 10/11 Equalizer** — Component EQ from iZotope's mastering suite.
- **Ozone 10/11 Dynamic EQ** — Frequency-reactive dynamic equalization.
- **Ozone 10/11 Match EQ** — Spectral matching between reference and target.
- **Solid EQ** — SSL-style EQ emulation from Native Instruments.

## Roland Hardware Emulation

A curated collection of Roland's Zenology and legacy hardware software emulations.

- **ZENOLOGY / ZENOLOGY FX** — Roland's software synth engine based on ZEN-Core, covering the full Roland synthesizer legacy.
- **D-50** — Emulation of the iconic 1987 Roland D-50 linear arithmetic synthesizer.
- **JD-800** — Software recreation of the classic 1991 programmable synthesizer.
- **JUNO-106 / JUNO-60 / JUNO-60 CHORUS** — Beloved analog polysynth emulations from Roland's Juno line.
- **JUPITER-4 / JUPITER-8** — Emulations of Roland's flagship vintage analog polysynths.
- **JV-1080** — 1994 ROM-based expander module emulation. Vast GM/XG sound library.
- **JX-3P** — Roland's 1983 DCO-based polysynth emulation.
- **SH-101 / SH-2** — Classic monophonic analog synth emulations.
- **SYSTEM-1 / SYSTEM-8 / SYSTEM-100** — Roland's ACB (Analog Circuit Behavior) synth emulations.
- **TB-303** — The iconic Roland bass synthesizer. Acid house foundation.
- **TR-606 / TR-707 / TR-727 / TR-808 / TR-909** — Roland's legendary drum machine lineup emulations.
- **PROMARS** — Roland's 1979 preset synthesizer emulation.
- **SRX Series** — Roland SRX expansion card emulations: Brass, Dance Trax, Electric Piano, Keyboards, Orchestra, Piano 1 & 2, Strings, Studio, World.
- **SOUND Canvas VA** — Roland's classic GM2 sound module emulation with hundreds of voices.
- **XV-5080** — Emulation of Roland's professional rackmount synthesizer module.
