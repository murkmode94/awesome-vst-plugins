# Contributing

Thanks for your interest in contributing to **Awesome VST Plugins**.

## What belongs here

An entry should satisfy at least one of these:

- It is a plugin with genuine production value (free or paid)
- It is an open-source audio tool worth studying or using
- It belongs to a recognized hardware emulation line
- It has lasting relevance in mixing, mastering, sound design, or composition

A plugin does **not** belong here just because it exists.

## What does not belong

- Abandonware with no update path and no community support
- Plugins with known stability or licensing issues
- Duplicate entries for the same plugin under different names
- Generic freeware dumps with no curation rationale

## Format

Each entry should follow this format:

```
- **Plugin Name** — Short, factual description. What does it do? What makes it notable?
```

- Use bold for the plugin name
- Keep descriptions to 1–2 sentences
- Link the official site or repo where relevant
- Place the entry in the correct category

## Categories

Check existing categories before adding. If a plugin spans multiple categories (e.g., a filter that is also a synth), place it in its primary use case and note the secondary use in the description.

## Submission

1. Fork this repository
2. Add your entry in the correct category, maintaining alphabetical order within each section
3. Open a pull request with a short description of what you are adding and why

## Code of conduct

Be direct. Be accurate. Don't promote.
