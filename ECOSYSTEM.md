# Ecosystem Alignment

This document defines where **awesome-vst-plugins** fits within the GareBear99 / TizWildin audio list network, and how to navigate between lists depending on what you need.

---

## What this repo is

**awesome-vst-plugins** is a producer- and engineer-facing curated collection of VST/VST3/AU plugins. It covers:

- Personal professional plugin collections (paid and free)
- Mixing, mastering, synthesis, effects, vocal processing, hardware emulation
- Organized by workflow category, not alphabetically or by vendor

It is **not** a developer resource list. It does not cover frameworks, DSP codebases, or build tooling. For that, see [awesome-audio-plugins-dev](https://github.com/GareBear99/awesome-audio-plugins-dev).

## How the two plugin lists relate

| | [awesome-audio-plugins-dev](https://github.com/GareBear99/awesome-audio-plugins-dev) | [awesome-vst-plugins](https://github.com/YOUR_USERNAME/awesome-vst-plugins) |
|---|---|---|
| **Focus** | Free & open-source | Personal collection (free + paid) |
| **Audience** | Developers and producers | Producers and engineers |
| **Frameworks/DSP** | Yes | No |
| **Hardware emulation** | Partial | Full (Roland, UAD, Waves) |
| **Sample packs** | TizWildin packs | Links to TizWildin packs |
| **Overlap** | Where a plugin is both free and production-worthy | Same |

Plugins that appear in **both** lists are doubly endorsed — they are worth both using and studying.

## Free tools from this ecosystem

Before spending money, check [awesome-audio-plugins-dev](https://github.com/GareBear99/awesome-audio-plugins-dev) for free alternatives. Notable free options that compete with paid plugins in this list:

| Free (TizWildin ecosystem) | Comparable paid plugin in this list |
|---|---|
| [FreeEQ8](https://github.com/GareBear99/FreeEQ8) | FabFilter Pro-Q 4, Ozone 11 EQ |
| [MeterMaid](https://github.com/GareBear99/MeterMaid) | Ozone Imager 2, Youlean, SPAN |
| [FreeVox8](https://github.com/GareBear99/FreeVox8) | VocalSynth 2, Nectar 3 |
| [Voxel Audio](https://github.com/GareBear99/Voxel_Audio) | SPAN, MiniMeters |
| [Therum](https://github.com/GareBear99/Therum_JUCE-Plugin) | Vital, Phase Plant |
